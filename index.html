pc -color- Set pen color
  The argument -color- can be one of:
  1. Neutrals: "black", "grey", or "white"
  2. Primaries: "red", "green", or "blue"
  3. Secondaries: "cyan", "magenta", or "yellow"
  4. A hexcode, like $469
 
Other Commands
goto X Y Go to position (X, Y), drawing a line if pen is down
make -name- -expression-     Set a variable to a value.
 
  1. The name of the variable is -name-
  2. The value of the variable is -expression-

The expression can be one of two things:

  1. A constant, such as 2
  2. A single-operator expression, like 4 + 5

The variables can be accessed with the colon (:) operator

                        make a 10
                        make b 12
                        make sum :a + :b

repeat N [...] Repeat a set of instructions N times.
 

For instance, the following code draws a square of side 40 points: 

                        repeat 4 [fd 40 rt 90]

The instructions to be repeated are placed one after another inside the square brackets. No separator is necessary. In this case, there are two instructions: an fd 40 to draw a line of 40 points, and a rt 90 to turn clockwise at a 90¬∞ angle.
